## 📱 Инструкции по настройке



### 1. v2rayNG (Android)
1. Скопируйте **Raw-ссылку** выше.
2. Откройте приложение → Нажмите на иконку **"Гамбургер"** (три полоски) в левом верхнем углу.
3. Выберите пункт **"Группы подписок"** (Subscription group setting).
4. Нажмите **"+"** → В поле `Remark` введите `Gh0st` → В поле `Optional URL` вставьте скопированную ссылку.
5. Нажмите на **галочку** в углу для сохранения.
6. Вернитесь на главный экран → Нажмите **три точки** (справа вверху) → **"Обновить подписку"** (Update subscription).

### 2. Hiddify / Happ (Multi-platform)
1. Нажмите на кнопку **"Новый профиль"** (New Profile) или иконку **"+"**.
2. Выберите **"Добавить по ссылке"** (Add from URL).
3. Вставьте скопированную ссылку в поле адреса.
4. Нажмите **"Добавить"** (Add).
5. Приложение само определит тип и загрузит все рабочие сервера `Gh0st_WhiteList`.

### 3. V2Box (iOS / iPhone)
1. Перейдите на вкладку **"Configs"** в нижней панели.
2. Нажмите **"+"** в правом верхнем углу.
3. Выберите **"Add Subscription"**.
4. В поле `Name` введите `Gh0st`, в поле `URL` вставьте ссылку.
5. Нажмите **"Add Subscription"**.
6. Нажмите кнопку **"Update"** (круговая стрелка), чтобы загрузить список серверов.

## 🛠️ Как это работает?
Каждый час наш алгоритм выполняет следующие действия:
* 📥 **Сбор:** Собирает конфиги из 5+ топовых мировых источников.
* 🔍 **Фильтрация:** Проверяет каждый сервер на доступность (Ping).
* 🏷️ **Брендирование:** Удаляет старые названия и присваивает каждому серверу флаг страны и номер.
* 🧹 **Чистка:** Удаляет дубликаты и нерабочие узлы, оставляя только "белый список".
