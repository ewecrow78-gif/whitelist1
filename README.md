## 💎 Особенности нашего листа
* **No Dead Nodes:** Фильтрация через Python-скрипт удаляет нерабочие сервера автоматически.
* **Smart Renaming:** Все сервера пронумерованы и имеют флаги стран для удобства.
* **Low Latency:** В приоритете сервера с минимальным временем отклика TCP.
* **Open Source:** Весь процесс сбора и очистки прозрачен и автоматизирован через GitHub Actions.
